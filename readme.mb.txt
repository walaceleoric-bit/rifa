# 🎟️ Sistema Profissional de Rifas

Sistema completo para gerenciamento de rifas online, desenvolvido para facilitar a organização, venda e controle de números de forma simples e eficiente.

Este sistema é ideal para:

✔ Igrejas  
✔ Sorteios beneficentes  
✔ Rifas online  
✔ Promoções  
✔ Eventos  

---

# 🚀 Funcionalidades do Sistema

✅ Cadastro de participantes  
✅ Controle automático de números da rifa  
✅ Painel administrativo completo  
✅ Configuração da rifa e prêmio  
✅ Histórico de participações  
✅ Sorteio ao vivo  
✅ Interface simples e moderna
